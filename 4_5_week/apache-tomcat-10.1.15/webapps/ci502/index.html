Very basic Servlets

  • hello The HelloWorld servlet that generates plain text.

  • my servlet My Servlet that generates HTML, accessed via the URL given in the @WebServlet annotation in the Java source code.

  • show-params The Show Parameters Servlet, showing parameters supplied by a browser in the http request object.

  • img servlet Shows an image that is read as a binary data and writes it through a HTTP response
